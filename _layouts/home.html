About Project

MIA_Visual (Model Intuitive Assessment_Visualization) is an open source project. ‘Learning curve analysis visualization’ helps determine if this model is reliable. ‘The visualization of the weight-update process’ helps to intuitively determine how the weights are being updated gradually. Through the ‘visualization of classification model performance evaluation indicators’, it is possible to help intuitively understand each evaluation indicator and to help select the appropriate model.

Features

Check our Features

We have 3 Features of Project

Visualize the learning curve

When checking the complexity of the model, we always have to draw the learning curve directly using a visualization tool. We will solve it by functioning it.

Visualize the machine learning

In order to visually check the update of model learning, there is a problem of having to draw a graph directly by storing the weights separately. We will solve it by functioning it.

Visualize classification model performance evaluation

It is not intuitive to visually compare numbers, which are indicators of the classification model. We graph this number so that we can check it intuitively.
